代表的なマークアップ言語

  • HTML (HyperText Markup Language): ウェブページの構造やデザインを記述し、ウェブブラウザで表示するために使用されます。
  • XML (eXtensible Markup Language) :データの構造や意味を定義するために使われます。設定ファイルやデータ交換用フォーマットとして広く利用されており、タグは自由に定義できます。
  • XHTML :HTMLの厳密なバージョンで、XMLの文法規則を適用したものです。より厳密なルールに基づいています。
  • Markdown :シンプルな記法で文章の構造を指定でき、学習しやすく読みやすいのが特徴です。Markdownファイルは、HTMLやPDFなどへ変換して利用されます。
  • TeX :学術分野で、図表や数式を含む印刷物を作成する際に、レイアウトを意図通りに整えるために使われます。
