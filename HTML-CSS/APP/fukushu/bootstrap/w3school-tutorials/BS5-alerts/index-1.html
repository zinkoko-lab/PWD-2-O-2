Alerts

Alerts are created with the .alert class, followed by a contextual color classes:

Success! This alert box could indicate a successful or positive action.
Info! This alert box could indicate a neutral informative change or action.
Warning! This alert box could indicate a warning that might need attention.
Danger! This alert box could indicate a dangerous or potentially negative action.

You can also add links inside the alerts with the .alert-link class:

Success! This alert box could indicate a successful or positive action. This is a link. Give it a click if you like.
Info! This alert box could indicate a neutral informative change or action. This is a link. Give it a click if you like.

Alerts can also be closed with the .alert-dismissible class:

Success! This alert box could indicate a successful or positive action.
